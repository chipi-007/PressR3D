Impresora de Resina

Este tipo de impresora 3D, utiliza resina líquida fotosensible como material de impresión. Este tipo de impresoras son ideales para obtener detalles muy precisos y superficies suaves. Sin embargo, requieren un poco más de cuidado y necesita postprocesamiento, para endurecerse completamente.

Impresora de Filamento

Para esta impresora, se utiliza filamento, generalmente de materiales como PLA, ABS o PETG, se calienta hasta fundirse. Ofrecen menos detalle y suavidad en comparación con las impresoras de resina, aunque son ideales para proyectos más grandes, funcionales o menos exigentes en precisión
